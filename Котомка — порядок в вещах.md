
Политика конфиденциальности

Котомка — порядок в вещах

Настоящая Политика конфиденциальности определяет порядок сбора, использования, хранения и защиты персональных данных пользователей мобильного приложения «Котомка — порядок в вещах» (далее — «Приложение»).

Используя Приложение, пользователь подтверждает согласие с условиями настоящей Политики конфиденциальности.

1. Какие данные мы собираем
1.1. Данные, предоставляемые пользователем

При регистрации и использовании Приложения пользователь может предоставлять следующие данные:

адрес электронной почты

данные учетной записи (логин, пароль — в зашифрованном виде)

фотографии вещей, добавляемые пользователем

информация о вещах, включая (но не ограничиваясь):

наименование

категория

цена

дата покупки

дополнительные пользовательские заметки

1.2. Данные, собираемые автоматически

При использовании Приложения автоматически могут собираться:

IP-адрес

тип устройства и модель

версия операционной системы

рекламные идентификаторы устройства

данные о действиях пользователя в Приложении

данные об ошибках и технических сбоях

2. Цели обработки данных

Собранные данные используются исключительно для:

регистрации и авторизации пользователей

корректной работы функционала Приложения

хранения и отображения информации о вещах пользователя

синхронизации данных между устройствами

аналитики и улучшения качества Приложения

показа рекламы

выполнения требований законодательства

3. Фотографии и пользовательский контент

Фотографии и иные данные о вещах:

добавляются пользователем добровольно

используются только внутри Приложения

не публикуются и не передаются третьим лицам без законных оснований

Ответственность за загружаемый контент несёт пользователь.

4. Аналитика

В Приложении используются сторонние аналитические сервисы для сбора обезличенной статистики использования Приложения.

Аналитические данные не используются для прямой идентификации пользователя.

5. Реклама

В Приложении используется мобильная реклама Яндекс Рекламной Сети (Яндекс РСЯ).

Яндекс может использовать рекламные идентификаторы, cookie и аналогичные технологии для показа персонализированной или неперсонализированной рекламы.

Обработка данных осуществляется в соответствии с Политикой конфиденциальности Яндекса:
https://yandex.ru/legal/confidential/

Пользователь может управлять персонализацией рекламы в настройках своего устройства.

6. Хранение и защита данных

Персональные данные пользователей хранятся в базе данных, размещённой на защищённых серверах timeweb.cloud.

Для защиты данных применяются современные меры безопасности, включая:

шифрование

ограничение доступа

защиту от несанкционированного доступа

Данные хранятся только в течение срока, необходимого для целей их обработки.

7. Передача данных третьим лицам

Персональные данные могут передаваться третьим лицам только в следующих случаях:

для работы аналитических и рекламных сервисов

по требованию законодательства

для обеспечения функционирования Приложения

Продажа или передача персональных данных третьим лицам в коммерческих целях не осуществляется.

8. Права пользователя

Пользователь имеет право:

получить информацию о своих персональных данных

изменить или удалить данные

удалить учетную запись

отозвать согласие на обработку данных

ограничить использование рекламных идентификаторов через настройки устройства

9. Данные детей

Приложение не предназначено для использования лицами младше 13 лет.

Если будет обнаружено, что персональные данные ребенка были получены без согласия законного представителя, такие данные будут удалены.
